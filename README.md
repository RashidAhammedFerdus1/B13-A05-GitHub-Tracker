1️⃣ What is the difference between var, let, and const?

Ans-1: Var Let Const all are variable. Var can be re-declared and re-assigned and Let cannot be re-declared in the same scope but can be re-assigned and Const cannot be re-declared or re-assigned.

2️⃣ What is the spread operator (...)?

Ans-2: The spread operator (...) in JavaScript is used to expand or spread the elements of an array. It usually used to copy arrays, merge arrays or object. 

3️⃣ What is the difference between map(), filter(), and forEach()?

Ans-3: map() creates a new array by applying a function to every element.
       filter() creates a new array with only the elements that pass a condition.
       forEach() runs a function on every element of the array but don't return a new array.

4️⃣ What is an arrow function?

Ans-4: An arrow function is a shorter way to write a function in JavaScript using the => syntax.

5️⃣ What are template literals?

Ans-5: Template literals is a way to write strings in JavaScript using backticks ( ).
